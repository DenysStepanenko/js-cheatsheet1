Циклы и управление потоком в JavaScript

Циклы позволяют многократно выполнять блок кода. Управление потоком определяет порядок выполнения инструкций.

Цикл `for`

Классический цикл `for` используется, когда известно количество итераций.

// Вывод чисел от 0 до 4
for (let i = 0; i < 5; i++) {
  console.log(i);
}

Цикл `while`

Цикл `while` выполняется, пока заданное условие истинно.

let count = 0;
while (count < 3) {
  console.log(`Счетчик: ${count}`);
  count++;
}
// Вывод:
// Счетчик: 0
// Счетчик: 1
// Счетчик: 2

Цикл `do...while`

Цикл `do...while` похож на `while`, но гарантирует выполнение тела цикла хотя бы один раз.

let attempts = 0;
do {
  console.log(`Попытка ${attempts + 1}`);
  attempts++;
} while (attempts < 2);
// Вывод:
// Попытка 1
// Попытка 2

Цикл `for...of`

Цикл `for...of` используется для перебора итерируемых объектов (массивы, строки, Map, Set).

const colors = ["красный", "зеленый", "синий"];
for (const color of colors) {
  console.log(color);
}
// Вывод:
// красный
// зеленый
// синий

Цикл `for...in`

Цикл `for...in` используется для перебора свойств объекта.

const person = { name: "Анна", age: 28 };
for (const key in person) {
  console.log(`${key}: ${person[key]}`);
}
// Вывод:
// name: Анна
// age: 28

Управление потоком (`break` и `continue`)

`break` используется для досрочного выхода из цикла, а `continue` - для пропуска текущей итерации.

for (let i = 0; i < 10; i++) {
  if (i === 3) {
    continue; // Пропустить итерацию, когда i равно 3
  }
  if (i === 7) {
    break; // Выйти из цикла, когда i равно 7
  }
  console.log(i); // Выведет: 0, 1, 2, 4, 5, 6
}
