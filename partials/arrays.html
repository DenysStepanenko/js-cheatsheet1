Массивы в JavaScript

Массивы - это упорядоченные коллекции данных. Они позволяют хранить несколько значений в одной переменной.

Создание массивов

Массивы можно создавать несколькими способами:

// Литерал массива (предпочтительный способ)
const fruits = ["Яблоко", "Банан", "Апельсин"];

// С помощью конструктора Array
const numbers = new Array(1, 2, 3, 4, 5);

// Создание пустого массива
const emptyArray = [];

Доступ к элементам

Доступ к элементам массива осуществляется по их индексу (начиная с 0):

const fruits = ["Яблоко", "Банан", "Апельсин"];

console.log(fruits[0]); // "Яблоко"
console.log(fruits[1]); // "Банан"

// Изменение элемента
fruits[1] = "Груша";
console.log(fruits); // ["Яблоко", "Груша", "Апельсин"]

// Длина массива
console.log(fruits.length); // 3

Методы массивов

JavaScript предоставляет множество полезных методов для работы с массивами:

  • push() / pop(): Добавление/удаление элемента в конце.
  • unshift() / shift(): Добавление/удаление элемента в начале.
  • splice(): Добавление/удаление/замена элементов в любом месте.
  • slice(): Создание копии части массива.
  • concat(): Объединение массивов.
  • indexOf() / lastIndexOf(): Поиск индекса элемента.
  • includes(): Проверка наличия элемента.
  • forEach(): Перебор элементов.
  • map(): Создание нового массива на основе существующего.
  • filter(): Создание нового массива с отфильтрованными элементами.
  • reduce(): Сведение массива к одному значению.
  • sort(): Сортировка массива.
  • reverse(): Изменение порядка элементов на обратный.
// Пример использования map
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8]

// Пример использования filter
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // [2, 4]
