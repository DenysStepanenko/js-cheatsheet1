Объекты в JavaScript

Объекты - это неупорядоченные коллекции пар "ключ-значение". Ключи - это строки (или символы), а значения могут быть любого типа данных.

Создание объектов

Объекты можно создавать с помощью литералов или конструктора.

// Литерал объекта (предпочтительный способ)
const person = {
  name: "Алексей",
  age: 35,
  city: "Екатеринбург",
  isAdmin: false,
  greet: function() { // Метод объекта
    console.log(`Привет от ${this.name}`);
  }
};

// С помощью конструктора Object
const car = new Object();
car.brand = "Toyota";
car.model = "Camry";
car.year = 2022;

Доступ к свойствам

Доступ к свойствам объекта осуществляется через точку или квадратные скобки.

console.log(person.name); // "Алексей" (через точку)
console.log(person["age"]); // 35 (через квадратные скобки)

// Использование переменной для доступа к свойству
const propertyName = "city";
console.log(person[propertyName]); // "Екатеринбург"

// Вызов метода
person.greet(); // "Привет от Алексей"

Изменение объектов

Свойства объектов можно добавлять, изменять и удалять.

// Добавление свойства
person.job = "Инженер";

// Изменение свойства
person.age = 36;

// Удаление свойства
delete person.isAdmin;

console.log(person);
/*
{
  name: "Алексей",
  age: 36,
  city: "Екатеринбург",
  greet: [Function: greet],
  job: "Инженер"
}
*/

Методы объектов

Объекты могут содержать функции, которые называются методами.

const calculator = {
  a: 0,
  b: 0,
  setValues(valA, valB) {
    this.a = valA;
    this.b = valB;
  },
  add() {
    return this.a + this.b;
  },
  subtract() {
    return this.a - this.b;
  }
};

calculator.setValues(10, 5);
console.log(calculator.add()); // 15
console.log(calculator.subtract()); // 5
