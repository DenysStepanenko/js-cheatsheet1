Типы данных

  • boolean — логический (`true` / `false`)
  • string — строка (текст)
  • number — числа (целые и дробные)
  • null — намеренное отсутствие данных
  • undefined — переменная не имеет значения
  • symbol — уникальный идентификатор
  • bigint — очень большие целые числа

Логические операторы

  • ! — НЕ (инверсия), пример: !true → false
  • || — ИЛИ, вернёт true, если хотя бы одно true
  • && — И, вернёт true, если все условия верны

Циклы

Цикл for


for (let i = 0; i < 5; i++) {
    console.log(i);
}

Цикл while


let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}

Цикл do...while


let i = 0;
do {
    console.log("Count: " + i);
    i++;
} while (i < 5);

Функции

Сумма чисел от 1 до N


function calculateTotal(number) {
    let total = 0, i = 1;
    while (i <= number) {
        total += i;
        i++;
    }
    return total;
}

Сумма чётных чисел


function calculateEvenTotal(number) {
    let sum = 0;
    for (let i = 1; i <= number; i++) {
        if (i % 2 === 0) {
            sum += i;
        }
    }
    return sum;
}

Методы строк

  • startsWith() — проверяет, начинается ли строка с указанного текста
  • endsWith() — проверяет, заканчивается ли строка указанным текстом
  • includes() — проверяет, содержится ли подстрока в строке
  • slice() — вырезает часть строки
  • toUpperCase() / toLowerCase() — изменение регистра
  • trim() — убирает пробелы по краям
