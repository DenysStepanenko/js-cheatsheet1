Типы данных

  • boolean — true или false. Пример: let isActive = true;
  • string — Текст. Пример: "Hello".
  • number — Числа. Пример: 42, -3.14.
  • null — Ничего. Пример: let value = null;
  • undefined — Не определено. Пример: let x;
  • symbol — Уникальный идентификатор. Пример: let id = Symbol("id");
  • bigint — Большие числа. Пример: 123n.

let isActive = true;
console.log(isActive);

Строки в JavaScript

Строка — это текст в кавычках. Она неизменяема.

Конкатенация

"Hello" + " " + "World" → "Hello World".

Шаблонные строки


const name = "Alex";
console.log(`Привет, ${name}!`);

Операторы сравнения

  • >, <, >=, <= — больше, меньше, больше/меньше или равно.
  • == — нестрогое равенство. Пример: 5 == "5" → true.
  • === — строгое равенство. Пример: 5 === "5" → false.

Math

  • Math.floor(3.7) → 3.
  • Math.random() → случайное число от 0 до 1.

console.log(Math.floor(3.7));

Основы функций

Функция — это блок кода, который можно вызывать.


function sayHello(name) {
    return `Привет, ${name}!`;
}
console.log(sayHello("Alex"));

Разветвления

Выполняют код в зависимости от условий.

if...else


let age = 18;
if (age >= 18) {
    console.log("Совершеннолетний");
} else {
    console.log("Несовершеннолетний");
}

Логические операторы

  • && — И (true, если оба верны).
  • || — ИЛИ (true, если хотя бы одно верно).
  • ! — НЕ. Пример: !true → false.

console.log(true && false);

Методы строк

  • includes("text") — проверяет наличие подстроки.
  • toLowerCase() — в нижний регистр.
  • slice(0, 3) — вырезает часть.

console.log("Hello".includes("ll"));

Циклы

while


let i = 0;
while (i < 3) {
    console.log(i);
    i++;
}

for


for (let i = 0; i < 3; i++) {
    console.log(i);
}

Массивы

Массив — список элементов.


const fruits = ["Яблоко", "Банан"];
console.log(fruits[0]);

Методы массива

  • push() — добавляет в конец.
  • pop() — убирает последний.

const numbers = [1, 2];
numbers.push(3);
console.log(numbers);

Итерация по массиву

forEach


[1, 2, 3].forEach(item => console.log(item));

Стрелочные функции


const add = (a, b) => a + b;
console.log(add(2, 3));
